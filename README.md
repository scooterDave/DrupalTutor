# DrupalTutor

A standalone React quiz application that tests your **Drupal 10 & 11** knowledge with 15 multiple-choice questions.

## Topics covered

- Composer & project setup
- Configuration Management
- JSON:API & REST
- PHP version requirements
- Themes (Claro, Olivero)
- Hooks & Form API
- Entity API
- Drush commands
- Migrations
- Views
- CKEditor 5 (Drupal 11)
- Services & Dependency Injection
- Routing
- Asset Libraries

## Getting started

```bash
npm install
npm start
```

## Build & deploy

```bash
npm run build
```

The `netlify.toml` at the repo root configures automatic deploys on Netlify with SPA redirect support.

## Tech stack

- [React 18](https://react.dev/)
- [Create React App](https://create-react-app.dev/)
- Plain CSS (Drupal blue `#0678BE` theme)
- Netlify (hosting)
